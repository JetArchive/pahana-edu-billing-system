Help and Support

Getting Started

Welcome to the Pahana Edu Bookshop Management System. This system helps you manage customers, inventory, and billing efficiently.

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection for accessing the system
  • Valid login credentials

User Guide

1. Login

To access the system, you need to log in with your username and password. If you don't have an account, please contact the system administrator.

2. Dashboard

The dashboard provides an overview of your bookshop operations, including customer count, item count, and recent activity.

3. Customer Management

In this section, you can:

  • Add new customers with their details
  • View all customers in the system
  • Edit existing customer information

4. Item Management

In this section, you can:

  • Add new items to your inventory
  • View all items in the system
  • Edit existing item information
  • Delete items from inventory

5. Billing System

In this section, you can:

  • Generate bills for customers
  • View all bills generated
  • Print bills for customers

Frequently Asked Questions

How do I reset my password?

Contact the system administrator to reset your password.

How do I add a new customer?

Go to the Customer Management section, click on the "Add Customer" tab, and fill in the required details.

How do I generate a bill?

Go to the Billing System section, select a customer, choose an item, enter the units, and click "Generate Bill".

Can I delete a customer?

Currently, the system does not support deleting customers. You can only edit their information.
